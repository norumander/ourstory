# UI Redesign — Warm Editorial Design System

## Direction
Warm editorial with personal touches. Literary, human, story-forward — like a well-designed magazine about generosity. Professional craft, not storybook whimsy.

## Color Palette

| Token | Hex | Usage |
|---|---|---|
| Primary (Warm Plum) | #5c3d6e | CTAs, links, progress bars, logo accent |
| Primary Light | #7a5490 | Hover states, gradient endpoints |
| Terracotta | #c47d3c | Secondary accent, AI section tabs, donation amounts |
| Gold | #d4a574 | Avatar rings, decorative borders, subtle accents |
| Deep Brown | #2a1f14 | Headlines, primary text |
| Warm Gray | #7a6555 | Body text |
| Muted | #a08b78 | Metadata, timestamps, labels |
| Border | #e8ddd4 | All borders (replaces gray-200) |
| Canvas | #faf7f2 | Page background (replaces gray-50) |
| Card Surface | #ffffff | Card backgrounds |
| AI Gradient From | #f5f0eb | AI section background start |
| AI Gradient To | #ede5db | AI section background end |

## Typography

- **Headlines**: Lora (serif) — Bold for page titles, Semibold for sections
- **AI narratives & quotes**: Lora Italic — pull-quote treatment
- **Donation messages**: Lora Italic — elevates user words
- **Body text**: Inter — all functional text, descriptions
- **UI elements**: Inter — forms, buttons, navigation, stats
- **Rule**: Lora = storytelling, Inter = functional

## Signature Design Moves

1. **Cream canvas** (#faf7f2) replaces cold gray-50 globally
2. **Warm borders** (#e8ddd4) replace gray-200 throughout
3. **Title-on-image hero** on fundraiser/community pages — cinematic overlay
4. **AI sections as editorial pull-quotes** — warm gradient bg, terracotta accent tab, Lora italic text, "✦ AI-generated insight" label
5. **"Words of support"** label instead of "Donations"
6. **Gold avatar rings** on user initials/avatars
7. **Lora serif headlines** throughout — editorial identity
8. **Plum gradient progress bars** — subtle depth
